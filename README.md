# 金融营销大师养成记

通过游戏化学习方式，掌握金融产品营销实务的核心技能！

## 功能特点

- 🎮 关卡系统 - 循序渐进的学习路径
- 📚 知识学习 - 系统化的知识点讲解
- ✍️ 答题训练 - 多种题型，即时反馈
- 🎖️ 成就系统 - 积分、等级、勋章奖励
- 🎵 音效反馈 - 答题和奖励都有音效
- 🎨 响应式设计 - 适配各种设备

## 快速开始

### 本地开发

直接用浏览器打开 `index.html` 即可使用！

### 部署到Cloudflare Pages

1. 访问 https://pages.cloudflare.com
2. 连接你的GitHub仓库 `https://github.com/cao-lg/course-game`
3. 配置构建设置：
   - 项目名称：`financial-marketing-master`（或你喜欢的名称）
   - 生产分支：`main`
   - 构建命令：（留空，不需要构建）
   - 构建输出目录：（留空，或填`/`）
4. 点击"Save and Deploy"！
